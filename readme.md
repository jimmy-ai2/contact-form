# Modern Contact Form

A visually striking and responsive contact form built with a mobile-first approach.  
It combines a clean dark-mode UI with intuitive inputs and smooth interactions to enhance user engagement.

---

## Features

- **Dark Mode Aesthetic**  
  Clean and modern color palette for a premium look.

- **Comprehensive Inputs**  
  Includes fields for:
  - Name  
  - Email  
  - Phone (optional)  
  - Subject  
  - Message  

- **Icon-Enhanced UI**  
  Font Awesome icons improve usability and visual clarity.

- **Interactive Feedback**  
  Smooth hover and focus effects with glowing input highlights.

- **Responsive Design**  
  Adapts seamlessly across mobile and desktop devices.

- **Animated Entry**  
  Subtle fade-in animation on page load.

---

## Tech Stack

- **HTML5** – Semantic structure  
- **CSS3** – Styling with:
  - Flexbox & Grid  
  - CSS Variables  
  - Transitions & animations  
- **Font Awesome** – Icons  
- **Google Fonts** – Clean typography

## How to Use

1. **Integrate**  
   Copy the HTML and CSS into your project.

2. **Fill the Form**  
   Enter details into the input fields.

3. **Submit**  
   Click the *Send Message* button.

4. **Customize**  
   Modify the primary color using CSS variables.
